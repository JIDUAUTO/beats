[[parse_accesslog]]
=== Generate a parse_accesslog of an event

++++
<titleabbrev>parse_accesslog</titleabbrev>
++++


[source,yaml]
-----------------------------------------------------
processors:
  - parse_accesslog:
      field: "message"
      ignore_missing: true
      time_field: "@timestamp"
      timezone: 'Asia/Shanghai'
      layouts:
        - '2006-01-02 15:04:05.000'
-----------------------------------------------------
